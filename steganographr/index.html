CoVerT

Hide text in plain sight using invisible zero-width characters. It’s digital steganography made simple.

Hide

Reveal the secret message hidden within a public message.

How it works

CoVerT works by converting your private message into binary data, and then converting that binary data into zero-width characters (which can then be hidden in your public message).

These characters are used:

  • WORD JOINER (U+2060)
  • ZERO WIDTH SPACE (U+200B)
  • ZERO WIDTH NON-JOINER (U+200C)

CoVerT is open source and available on GitHub.
